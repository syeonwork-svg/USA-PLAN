# ✈️ 미국 20일 여행 플래너 (USA Travel Planner)

Apple 캘린더 스타일의 극도로 미니멀하고 직관적인 미국 서/동부 20일 여행 계획 및 경비 관리 웹 애플리케이션입니다.

## ✨ 주요 기능
1. **캘린더 뷰 (메인)**: 사용자가 업로드한 계획표 스크린샷의 모든 레이아웃(애틀랜타, 올랜도, 마이애미, 뉴욕 등의 도시 배지, 로드트립 🚗 아이콘, 숙박 기간 스팬 바)을 완벽 구현하였습니다.
2. **날짜별 여행 계획 (Timeline)**: 20일간의 매칭 타임라인을 제공하여 활동 추가/수정/삭제가 자유롭습니다.
3. **경비 관리 (Budget)**: 카테고리별 경비 리스트와 동적인 SVG 도넛 차트 피드백을 제공합니다.
4. **지도 연동 (Maps)**: Leaflet.js를 기반으로 API 키 없이 Apple Maps 스타일(Muted Gray)의 지도를 제공하며, 설정에서 Google Maps API Key 입력 시 구글 맵으로의 동적인 전환을 지원합니다.
5. **라이트 / 다크 모드**: OS 환경설정에 동기화 및 수동 토글이 가능합니다.
6. **로컬 영구 저장**: 브라우저의 `localStorage`를 사용하여 페이지를 새로고침하거나 종료하더라도 데이터가 그대로 저장됩니다.

## 📂 파일 구조
* `index.html`: 메인 웹페이지 구조 및 모달 팝업 레이아웃
* `css/style.css`: Apple 스타일(SF Pro 폰트, Glassmorphism, 부드러운 전환 효과) 및 반응형 미디어 쿼리 스타일시트
* `js/mockData.js`: 캡처본과 100% 일치하는 20일간의 초기 모의 일정 및 경비 데이터
* `js/storage.js`: LocalStorage 읽기/쓰기 유틸리티
* `js/calendar.js`: 캘린더 날짜 계산 및 일정 렌더링 제어
* `js/itinerary.js`: 날짜별 수직 타임라인 및 CRUD 제어
* `js/expenses.js`: 경비 예산 현황 및 커스텀 SVG 차트 렌더링
* `js/map.js`: Leaflet 및 Google Maps 듀얼 맵 모듈, 경로 렌더링
* `js/app.js`: 전체 앱 초기화 및 화면 제어 컨트롤러

## 🚀 로컬 실행 방법

프로젝트 폴더 내에서 다음 명령어를 실행하여 웹 서버를 띄운 뒤 브라우저에서 접속하실 수 있습니다.

```bash
# 1. npm을 사용한 실행 (8080 포트)
npm start

# 또는 터미널에서 파이썬 빌트인 서버 이용 시:
python3 -m http.server 8080
```

서버가 실행되면 크롬이나 사파리 브라우저에서 `http://localhost:8080`에 접속하여 사용하실 수 있습니다.
